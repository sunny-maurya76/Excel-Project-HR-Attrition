# HR Attrition Analysis Project

##  Project Overview
This project analyzes **employee attrition** using Excel.  
The goal is to **identify factors leading to employee turnover** and provide actionable insights for HR decision-making.

---

##  Key Features
- Data cleaning and preprocessing in Excel  
- Employee attrition trends and patterns analysis  
- Visualization using **charts and pivot tables**  
- Insights on factors like **department, salary, job satisfaction, and work hours**
